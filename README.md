# AI Cancer Cell Prediction System

An AI-powered machine learning project designed to predict the likelihood of cancer based on patient medical data.  
This project was developed using an **Agile Sprint-based workflow** consisting of **3 sprints**: Data Preparation, Model Development, and Model Optimization.

---

## 📌 Project Overview

Cancer prediction using machine learning can assist medical professionals by identifying patterns in patient data and predicting potential cancer risks.

This project builds a **complete machine learning pipeline**, including:

- Data generation and preprocessing
- Feature engineering
- Model training
- Model optimization
- Model evaluation and prediction visualization

# ⚙️ Machine Learning Workflow

The project follows a **structured machine learning workflow**:

5. Performance Evaluation
6. Prediction Visualization

---

# 🧩 Sprint-Based Development

## Sprint 1 – Data Preparation

Tasks completed:

- Generated **1000 synthetic patient records**
- Introduced data anomalies:
  - Null values
  - Duplicates
  - Outliers
  - Incorrect formats
- Cleaned dataset using:
  - **IQR Method**
  - **Z-Score**
  - **Data Imputation**
- Created **System Requirement Specification (SRS)**
- Developed **Project Management Plan**

Deliverables:

- Data preprocessing Python script
- Cleaned dataset
- SRS documentation
- Project plan

---

## Sprint 2 – Model Development

Tasks completed:

- Feature engineering
- Dataset partitioning (**80/20 train-test split**)
- Model training using **Random Forest Classifier**
- Model evaluation using performance metrics
- Prediction visualization

Evaluation Metrics:

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

Deliverables:

- Model development code
- Processed dataset
- Model metrics report
- Prediction report
- Sprint documentation

---

## Sprint 3 – Model Optimization & Final Delivery

Tasks completed:

- Applied **PCA (Principal Component Analysis)** for dimensionality reduction
- Performed **Hyperparameter Tuning using GridSearchCV**
- Retrained optimized model
- Conducted final evaluation and analysis
- Generated prediction charts and interpretation report

Final Evaluation Metrics:

- Macro Average
- Weighted Average
- F1 Score
- Confusion Matrix

Deliverables:

- Optimized model code
- Final prediction dataset
- Interpretation report
- Final project presentation

# 📊 Model Used

The primary model used in this project is:

**Random Forest Classifier**

Key advantages:

- Handles high-dimensional data
- Reduces overfitting through ensemble learning
- Provides strong classification performance

Optimization Techniques:

- PCA (Dimensionality Reduction)
- GridSearchCV (Hyperparameter Tuning)
